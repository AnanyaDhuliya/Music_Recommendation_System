# Music_Recommendation_System
Music Recommendation System developed as a college minor project using Python, machine learning concepts, and AI-assisted development.

🎓 About This Project

Raagify is a Bollywood Music Recommendation System developed as my Minor Project during the second year of my Computer Science degree.

The goal of this project was to understand how recommendation systems work and how machine learning concepts can be applied in real-world applications. Through this project, I explored data processing, similarity-based recommendations, API integration, and full-stack web development using Python and Flask.

The application allows users to search for Bollywood songs and receive recommendations based on song characteristics such as genre, mood, and tempo. It also enhances the user experience by displaying album artwork and audio previews.

 Key Learning Outcomes

* Understanding recommendation systems and similarity-based algorithms
* Working with machine learning libraries such as Scikit-learn
* Building REST APIs using Flask
* Processing and analyzing datasets using Pandas
* Integrating third-party APIs into web applications
* Creating responsive user interfaces with HTML, CSS, and JavaScript
* Managing and organizing a complete full-stack project
